# @renaiss-shipflow/skill

Claude Code skill that wraps the [`@renaiss-shipflow/cli`](https://www.npmjs.com/package/@renaiss-shipflow/cli) for use inside Claude Code conversations.

## Install

```sh
npm install -g @renaiss-shipflow/cli
# Then install this skill into Claude Code (see Claude Code docs).
renaiss-shipflow login
```

See the friendly walkthrough at https://renaiss-shipflow.zeabur.app/docs/claude-skill for a graphical guide.
